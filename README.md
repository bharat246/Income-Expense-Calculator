# Income Expense Calculator

A simple web-based financial tracker built using **HTML, CSS, and JavaScript**.

## Features
- Add income and expense entries  
- Edit and delete entries  
- Filter records (All / Income / Expense)  
- LocalStorage support (data saved even after refresh)  
- Responsive design  
- Real-time calculation of:
  - Total Income  
  - Total Expense  
  - Net Balance  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## How to Use
1. Enter description and amount  
2. Select type (Income / Expense)  
3. Click **Add**  
4. Entries appear in the list  
5. Use edit/delete buttons for modifications  
6. Filters help view specific types  
7. Summary updates automatical
